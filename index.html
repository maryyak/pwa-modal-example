Progressive Web Application

Как установить приложение на ваше устройство?

  1. Когда появляется окно с предложением установки, нажмите кнопку "Установить приложение".
  2. После установки приложение будет доступно с вашего домашнего экрана, как любое другое приложение.

Как установить на устройство с iOS?

  • Для установки на устройствах iOS необходимо выполнить несколько шагов:
  • Откройте сайт в браузере Safari (не поддерживается в других браузерах).
  • Когда сайт загружен, нажмите на кнопку Поделиться (иконка внизу экрана) и выберите опцию Добавить на экран домой.
  • После этого приложение появится на вашем домашнем экране, и вы сможете использовать его как обычное мобильное приложение.

Почему у меня не появляется окно с предложением установки?

  • Приложение уже установлено: Если вы уже установили приложение на ваше устройство, предложение повторно появляться не будет.
  • Установка не поддерживается: Ваша версия браузера или операционная система могут не поддерживать установку приложения.

Если вы отклонили предложение, вы можете установить приложение позже, перезагрузив страницу и выбрав установку снова.
